## How to Install the Modeling Shortcuts
1. Open Excel → File → Options → Add-Ins → Go
2. Click "Browse" and select `ModelingShortcuts.xlam` from this folder
3. Done! Use Ctrl+Shift+1, 2, 5, 8 etc. across all Excel files
